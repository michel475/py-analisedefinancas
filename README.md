Projeto de analise de finanças utilizando Neo4J
*******************************************************
O projeto consiste em utilizar um LLM
para realizar análise de finanças pessoais
e extrair informação a partir da análise.

Tendo como base a informação que o usuário quer
o objetivo do projeto é permitir que o LLM
utilize o contexto da conversa/pergunta
para realizar análises precisas em relação à:
-Gastos
-Investimentos
-Recebimentos
***********
Basicamente análise da entrada e saída de capital.
